# KPI-DRIVE — Канбан-доска задач

Тестовое задание на позицию Flutter-разработчика.  
Веб-приложение на Flutter для управления задачами в формате канбан-доски с поддержкой перетаскивания и синхронизацией с API KPI-Drive.

---

## Стек технологий

| Технология               | Назначение            |
| ------------------------ | --------------------- |
| Flutter (Web)            | UI-фреймворк          |
| Riverpod (StateNotifier) | Управление состоянием |

## Реализованный функционал

**Отображение задач**

- Загрузка задач с API через POST-запрос с фильтрацией по периоду и типу
- Автоматическая группировка задач по `parent_id` — каждый `parent_id` становится отдельной колонкой
- Сортировка задач внутри колонки по полю `order`
- Индикатор загрузки при получении данных
- Кнопка обновления данных в верхней панели

**Перетаскивание (Drag & Drop)**

- Мгновенное перетаскивание без задержки (без `ReorderableListView`)
- Перемещение задач между колонками (изменение `parent_id`)
- Перестановка задач внутри одной колонки (изменение `order`)
- Визуальная подсветка целевой колонки при наведении
- Визуальный индикатор позиции вставки при сортировке внутри колонки
- Оптимистичное обновление UI — доска реагирует мгновенно, запрос идёт в фоне
- Автоматический откат состояния при ошибке сервера

**Синхронизация с сервером**

- Сохранение `parent_id` и `order` через отдельные POST-запросы
- Индикатор сохранения на карточке (spinner вместо порядкового номера)
- Обработка ошибок с отображением баннера и возможностью закрыть его

**Адаптивность**

- Ширина колонок адаптируется под размер экрана
- Горизонтальная прокрутка при большом количестве колонок

## Запуск

### Требования

- Flutter SDK >= 3.0.0
- Dart SDK >= 3.0.0

### Установка зависимостей

```bash
flutter pub get
```

### Запуск для разработки

Приложение работает только на Web. Из-за ограничений CORS браузера необходимо запускать Chrome с отключённой проверкой безопасности:

```bash
flutter run -d chrome --web-browser-flag "--disable-web-security"
```

> **Важно:** флаг `--disable-web-security` предназначен исключительно для разработки и тестирования. В продакшн-окружении CORS должен быть настроен на стороне сервера.

### Сборка для продакшн

```bash
flutter build web
```

## API

**Базовый URL:** `https://api.dev.kpi-drive.ru/_api`  
**Авторизация:** Bearer-токен в заголовке `Authorization`

### Получение задач

```
POST /indicators/get_mo_indicators
```

| Параметр          | Значение                                  |
| ----------------- | ----------------------------------------- |
| `period_start`    | `2026-04-01`                              |
| `period_end`      | `2026-04-30`                              |
| `period_key`      | `month`                                   |
| `requested_mo_id` | `42`                                      |
| `behaviour_key`   | `task,kpi_task`                           |
| `with_result`     | `false`                                   |
| `response_fields` | `name,indicator_to_mo_id,parent_id,order` |
| `auth_user_id`    | `40`                                      |

### Сохранение поля задачи

```
POST /indicators/save_indicator_instance_field
```

| Параметр             | Описание                           |
| -------------------- | ---------------------------------- |
| `indicator_to_mo_id` | ID задачи                          |
| `field_name`         | Имя поля (`parent_id` или `order`) |
| `field_value`        | Новое значение                     |
| `auth_user_id`       | `40`                               |

При перемещении задачи выполняются два последовательных запроса: сначала обновляется `parent_id`, затем `order`.

## Модель данных

```dart
class Task {
  final int indicatorToMoId; // Уникальный идентификатор задачи
  final int parentId;        // ID папки/колонки

  final int order;           // Порядковый номер внутри колонки
}
```
